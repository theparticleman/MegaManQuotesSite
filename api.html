Mega Man Quotes

You can use the API that powers this site in your own projects if you want to. It uses an HTTP endpoint that returns JSON.

The endpoint you need to send an HTTP GET to is

https://api.megamanquotes.com/random-quote

The endpoint will return a JSON object that looks something like this 

{
    'text': 'The main text of the quote',
    'author': 'The name of the author',
    'source': 'The source of the quote if available. Otherwise an empty string.'
}
